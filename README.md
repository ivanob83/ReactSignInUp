# ReactSignInUp

**ReactSignInUp** is a modern authentication module for React applications, built with **TypeScript**, **React Hook Form**, **Firebase Authentication**, **Redux Toolkit**, and **Material-UI / Tailwind CSS**. It provides fully functional sign-up, sign-in, forgot password, and reset password flows.

---

## Features

-   **Sign Up & Sign In**  
    Email/password authentication with form validation.

-   **Forgot & Reset Password**  
    Password recovery flow using Firebase's secure reset emails.

-   **Redux Integration**  
    Auth state persisted via `redux-persist` for auto-login on refresh.

-   **Social Login Buttons**  
    Supports Google and GitHub OAuth login.

-   **Responsive UI**  
    Clean, modular UI with Material-UI components and optional Tailwind styling.

-   **Error Handling**  
    Inline form validation and Firebase error messages.

---

## Tech Stack

-   React + TypeScript
-   React Hook Form
-   Firebase Authentication
-   Redux Toolkit + redux-persist
-   Bootstrap
-   React Router v6
